# WHS3_Secure_coding
**중고거래 플랫폼에 있어야 하는 기능:**

- 사람들이 플랫폼에 가입할 수 있어야 함
- 상품들을 올리고 볼 수 있어야 함.
- 플랫폼 사용자들끼리 소통이 가능해야함.
- 악성 유저나 상품을 차단 해야 함.

>>해당 부분에 대한 시스템 설계는, 24page에서 명세한 것을 모두 포함할 것
(필요에 따라 기능추가 할 것)

**24page에 명세한 내용**

유저 관리

- 회원 가입 기능
- 로그인 기능
- 사용자 조회 기능
- 마이페이지 기능 (소개글 및 비밀번호 업데이트)

상품 관리

- 상품 등록 기능
- 등록된 상품 관리 기능
- 상품 조회 기능
- 상품 상세 페이지 기능

유저 소통 기능

- 실시간 전체 채팅 기능
- 1대1 채팅 기능

악성 유저 필터링

- 불량 유저 및 상품 신고 기능
- 불량 상품 삭제 기능
- 불량 유저 휴면 기능

- 관리자가 플랫폼의 모든 요소를 관리할 수 있어야 함
